Bug #1 - resolved via CMSIS migration

Original code manually typed 0x40004408 for USART2_BRR. Correct address is 0x4000440C (offset 0x0C, not 0x08 - that is CR3). Baud divisor written to CR3 = no output, no error. Four hours of debugging, one wrong hex digit. Fix: moved to CMSIS struct access (USART2->BRR) - the compiler resolves the offset, no addresses to mistype.

Bug #2 - IWDG hung before main() printed anything

The IWDG SR update flags (PVU, RVU) only clear once the watchdog is running. Writing 0x5555 (unlock) before 0xCCCC (start) means they never clear - while(IWDG->SR != 0) loops forever. Also: LSI oscillator must be explicitly enabled via RCC->CSR |= RCC_CSR_LSION and LSIRDY confirmed before any IWDG config. Found with GDB - firmware appeared dead, no UART output at all. RM0351 section 34.3.

Architecture

Prototype phase: one file. Refactor phase: three layers. Phase 4: FreeRTOS two-task design - task_sensor reads, filters, detects BPM, kicks IWDG; task_uart blocks on a depth-1 queue and formats output. Rule: application never touches registers. Driver never calls application. No malloc anywhere in authored code.

Application
task_sensor (priority 2): sensors → filter → BPM → xQueueOverwrite → IWDG kick → vTaskDelay 500 ms. task_uart (priority 1): xQueueReceive blocks → UART format.

SysTick configured for 1 ms tick. Calibrated delay_ms() replaces busy-wait loop. IWDG with 4-second timeout using LSI oscillator. LSI must be enabled and LSIRDY confirmed before IWDG config - got this wrong first, debugged with GDB.

PHASE 3

Signal processing - BPM from raw PPG data.

✓ DONE
BPM detection from PPG signal

Dynamic threshold: midpoint of rolling min/max over 16-sample history window. Rising-edge crossing detection with 333 ms refractory period. BPM = 60000 / interval_ms. Shows --- until two crossings confirmed, then live reading. Verified 119 BPM on hardware.

ARM_CM4F port. IWDG kicked every 500 ms inside task_sensor. Peripheral drivers migrated to CMSIS device headers (stm32l476xx.h). Verified on hardware: TMP117 25.3 C, IR ~87000 finger contact, BPM 119.
